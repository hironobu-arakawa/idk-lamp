# Relationship with idk-lamp

**idk-lamp** is a concept positioned below the Decision Closure Structure.

- Decision Closure Structure: Decides whether to close the decision
- idk-lamp: Visualizes the state where it cannot be closed

idk-lamp is not the cause.
It is a **signal indicating the result of the structure**.

It indicates a state of:
"Not stopping because it doesn't know", but
"Not closing because it must not close".
